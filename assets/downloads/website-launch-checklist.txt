外贸网站上线检查清单
====================

一、品牌与内容
[ ] Logo、品牌名称和联系方式一致
[ ] 公司介绍、产品参数和认证信息已经确认
[ ] 产品图片清晰，并具有准确的替代文本
[ ] 每个核心页面都有明确标题和下一步行动
[ ] FAQ 回答真实采购问题，不使用无法验证的承诺

二、询盘路径
[ ] 邮箱、微信、WhatsApp 或表单至少有一个可用入口
[ ] 手机端可以完成咨询操作
[ ] 表单必填项、邮箱格式和错误提示正常
[ ] 提交后有明确反馈或后续说明
[ ] 已记录需要追踪的咨询按钮和表单事件

三、SEO 基础
[ ] 每页只有一个 H1
[ ] Title 和 Description 不重复
[ ] Canonical 指向正确正式网址
[ ] robots.txt 和 sitemap.xml 可以访问
[ ] sitemap.xml 不包含 404 或测试页面
[ ] 重要页面已加入站内导航和内链
[ ] 图片文件名、尺寸和替代文本合理
[ ] 结构化数据可以正常解析

四、技术与体验
[ ] HTTPS 正常且没有混合内容
[ ] 桌面端和手机端没有横向滚动
[ ] 图片、按钮和菜单没有遮挡正文
[ ] 所有内部链接均返回正常页面
[ ] 404 页面和失效外链已经处理
[ ] 键盘可以操作菜单、FAQ 和表单
[ ] 动画支持“减少动态效果”设置
[ ] JavaScript 加载失败时核心正文仍可阅读

五、数据与上线
[ ] GA4、GTM 或其他分析工具已正确安装
[ ] Google Search Console 已验证
[ ] 核心转化事件已经测试
[ ] 已记录上线日期和页面基线数据
[ ] 已安排上线后 7 天和 30 天复查
